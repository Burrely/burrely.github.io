Download the modpack here.

Installation instructions:

  1. Install CurseForge for your platform.
  2. Select Minecraft from the game selection Window.
  3. Create a custom profile (located in the top right of the Application).
  4. Select "import" at the top of the pop-up window.
  5. Create the modpack.
  6. Make sure that all the mods are enabled
    (There have been reports of mods not being enabled by default, if there are any issues make sure to check this first!).
  7. Run the modpack!
